# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-27

### Added
- **The Review Queue**: Batch processing for notifications in focused sessions.
- **Smart Actions & AI Triage**: Local Ghost AI parses notification context to generate dynamic replies and actions.
- **Daily Brief**: Summarized dashboard of what needs your attention daily.
- **Premium UX Redesign**: Custom glassmorphic motion design system, featuring physics-based spring animations and glow effects.
- **Privacy-First Storage**: Persistent, fully local SQLite database powered by Drift for on-device telemetry and data handling.

### Fixed
- **Zero-Lint Codebase**: Extensive cleanup of unused elements and deprecated flutter API usages.
- Resolved context synchronization across async gaps.

### Changed
- **Fully Optimized Build**: Size reductions and faster loading times with full release tree-shaking applied for production builds.
